# categorizer

Анализатор тональности текстов на русском языке с использованием модели RuBERT.

## Требования

- Python 3.8+
- pip или другой менеджер пакетов Python

## Установка

- Склонировать репозиторий, перейти в рабочую директорию
- Создайте виртуальное окружение (рекомендуется):`python -m venv venv`
- Активируйте виртуальное окружение
- Установите зависимости: `pip install -r requirements.txt`

## Запуск

Запустите сервер FastAPI:
```bash
uvicorn main:app --reload
```

Сервис будет доступен по адресу: `http://localhost:8000`

## API Endpoints

### GET /
Проверка статуса сервиса

### POST /sentiment
Анализ тональности текстов

**Пример запроса:**
```json
{
  "texts": [
    "Это отличный продукт!",
    "Очень плохое качество",
    "Нормально, ничего особенного"
  ]
}
```

**Пример ответа:**
```json
{
  "results": [
    {
      "text": "Это отличный продукт!",
      "sentiment": "positive",
      "confidence": 0.95
    },
    {
      "text": "Очень плохое качество",
      "sentiment": "negative",
      "confidence": 0.92
    },
    {
      "text": "Нормально, ничего особенного",
      "sentiment": "neutral",
      "confidence": 0.87
    }
  ]
}
```

## Модель

Используется предобученная модель **seara/rubert-tiny2-russian-sentiment** на основе RuBERT для классификации тональности на три класса: положительная (positive), отрицательная (negative) и нейтральная (neutral).
